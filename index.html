Code Example

Create a function that returns the sum of the two lowest positive numbers given an array of minimum 4 positive integers. No floats or non-positive integers will be passed. JavaScript 

function sumTwoSmallestNumbers(numbers) {
    const sort = numbers.sort((a, b) => a - b);
    return sort[0] + sort[1];
}

// Example usage:
console.log(sumTwoSmallestNumbers([5, 8, 12, 19, 22])); // 13
console.log(sumTwoSmallestNumbers([15, 28, 4, 2, 43])); // 6
